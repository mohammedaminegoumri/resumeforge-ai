# ResumeForge AI

**AI-powered resume optimizer that tailors your resume to any job description while preserving your original design 100%.**

→ [Read the full Deployment Guide](./DEPLOYMENT_GUIDE.md)

## Quick Start

```bash
npm install
# Add ANTHROPIC_API_KEY to .env.local
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/resumeforge-ai&env=ANTHROPIC_API_KEY&envDescription=Get%20your%20key%20at%20console.anthropic.com)
